Pemrograman Android


Dasar Pemrograman Android. Android merupakan sistem operasi yang berbasis linux yang di kembangkan untuk perangkat layar sentuh seperti smartphone ataupun komputer tablet. Sekarang sistem operasi ini semakin populer seiring dengan berkambangnya smartphone. Hampir semua penduduk bumi menggunakan smartphone untuk membantu bekerja ataupun hanya untuk sekidar sebagai hiburan. Sehingga semakin banyak pengembang aplikasi membidik sasaran pasar android, karena penggunanya selalu bertambah dari tahun ke tahun. Smartphone berbasis android merupakan trend terbesar abad ini.


Beberapa bagian yang mungkin harus anda pahamai untuk menjadi pengembang aplikasi berbasis android yaitu :

activity
activity merupakan container untuk user interface ( UI ), sedangkan sebuah aplikasi android dibangun dari satu atau beberapa activity.

Intent
Intent merupakan sistem pesan utama yang menjalankan Android. Intent terdiri dari Action yang harus dijalankan (Tampil, Ubah,Dial, dll) dan Data. Intent digunakan untuk memulai aktivitas dan komunikasi antar bagian dari sistem Android. Suatu aplikasi dapat mengirimkan atau menerima intent.

View
View merupakan elemen dasar UI (User Interface). Contoh : ContextMenu, Menu, View, SurfaceView.

Widgets
Widgets adalah elemen UI yang lebih canggih. Merupakan control untuk interaksi antara sistem dengan pengguna. Contoh : Button, CheckBox, DatePicker, DigitalClock, Gallery, FrameLayout, ImageView, RelativeLayout, PopupWindow.

Berikut tools yang digunakan untuk pembuatan aplikasi Android:

IDE Eclipse

Eclipse adalah IDE yang dipakai untuk pemrograman android. Khususnya bagi pemula di sarankan belajar dulu pake IDE Eclipse agar tidak mengalami kesulitan. Ada terkadang kebanyakan programer java datang dari NetBeans yang terkadang ketika beralih ke Android langsung pakai IDE Netbeans.

IDE Android Studio

Melihat perkembangan saat ini, Android Studio saat ini sudah menjadi official IDE untuk Android Development, artinya di web resmi Android Developer dikatakan kedepan nya direkomendasikan harus menggunakan IDE Android Studio.
